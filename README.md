# Weather App 🌤️

A real-time weather application built with React.js and OpenWeather API.

## 🔗 Live Demo
[View App](https://we-delta-lake.vercel.app)

## ✨ Features
- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- 💧 Humidity and wind speed info
- 📱 Fully responsive design

## 🛠️ Tech Stack
- React.js
- OpenWeather API
- CSS

## 🚀 Getting Started
```bash
npm install
npm run dev
```
